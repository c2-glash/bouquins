A la racine du projet se trouve le fichier webpack.config.js qu'il faut éditer.
Il faut décommenter la ligne .enableReactPreset(), normalement située à la ligne 70.
Voir les informations dans :
https://symfony.com/doc/current/frontend/encore/reactjs.html

// Dans le fichier webpack.config.js ajouter le code suivant, normalement à la ligne 72 :
.configureBabel(function(babelConfig) {
    babelConfig.plugins.push('@babel/plugin-proposal-class-properties');
});
